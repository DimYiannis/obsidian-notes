# Subnet

A network sliced into smaller networks. All devices sharing the same [[network-host-portion|network portion]] = same subnet ("sub-network"). Defined by an [[ip-address]] + a [[subnet-mask]] (or [[cidr]]).

Each subnet reserves a [[network-broadcast|network address and broadcast address]]. To leave the subnet, traffic goes to the [[default-gateway]].
